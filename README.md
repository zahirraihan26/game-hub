# 🎮 PlayVerse - A Game Library

**PlayVerse** is an engaging online platform where users can discover, explore, and support indie game developers. It allows users to browse through different games, view detailed information, and install their favorite ones.

---

## 🌐 Live Demo

🔗 **Live URL:** [https://enchanting-shortbread-d62d1c.netlify.app/](https://enchanting-shortbread-d62d1c.netlify.app/)

---

## 🧩 Project Purpose

The purpose of **PlayVerse** is to create a community-driven game library that supports independent developers and help
 gamers explore new and creative titles in a visually appealing and interactive way.

---

## 🚀 Key Features

- 🕹️ **Browse Games:** View all available games with details like name, category, rating, and developer.  
- 📖 **Game Details Page:** See in-depth information about each game.  
- 🔒 **User Authentication:** Login, Register, and Logout using Firebase Authentication.  
- 🔑 **Protected Routes:** Only logged-in users can view game details or profile pages.  
- 🧠 **Forget Password:** Users can reset their password using email verification.  
- 👤 **User Profile:** View logged-in user’s info and manage account settings.  
- 💬 **Dynamic Data:** Game and news data loaded dynamically from JSON files.  
- 🎨 **Responsive Design:** Fully responsive UI built with Tailwind CSS.  
- ⚡ **Animation Effects:** Smooth UI transitions using Framer Motion.  
- 🧭 **Routing:** Implemented using React Router DOM v6.  
- 🔥 **Firebase Integration:** Firebase used for authentication and hosting.
